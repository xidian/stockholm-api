# 通用推送接口

## 手机端发往设备端

* POST /mobile/push

requestHeaders
```
Access-Token: <your-access-token>
UUID: <your-device-uuid>
```

requestBody
```json
{
	"whisper": "your-content",
	"extras": {
		"order": "...",
		"packageName": "your package name",
		"timestamp": "true (optional)"
	}
}
```
extras里面的json写任何字段都行
