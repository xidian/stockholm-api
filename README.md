# stockholm-api

Add it in your root build.gradle at the end of repositories:
```
allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
  ```
  Add the dependency
  ```
  dependencies {
	        compile 'com.github.xidian:stockholm-api:0.1.0'
	}
  ```
